Anime checker is a tool that verifies the integrity of your files. 
It calculates the CRC32 of your files and compare it with the one in the filename. 
Useful with fansubbed anime. 


System Requirements
-------------------
  - Windows(R) 2000, Windows XP(TM) or higher. Linux.
  - GTK (linux only).


How to install
--------------
 unzip all the files into a directory.


Files
-----
animechecker      : gui tool to calculate the crc.
acc               : command line version of the tool.
install-shortcuts : create/remove shorcuts (Windows).
